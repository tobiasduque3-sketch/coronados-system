# Report Site

This folder is a standalone static site for the Superapp campaign report.

## Files

- `index.html`: public landing page for the report
- `superapp-campaign-report.pdf`: downloadable PDF version

## Deploy

Any static host can publish this folder directly.

### GitHub Pages

1. Push the repository to GitHub.
2. In repository settings, open `Pages`.
3. Set the source to `Deploy from a branch`.
4. Choose your main branch and the `/docs` folder.

### Netlify or Vercel

- Import the repository.
- Set the publish/output directory to `docs`.
- No build command is required.
